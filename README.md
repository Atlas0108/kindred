## Firebase / Firestore

The Dart package name remains **kindred** (internal `package:kindred/...` imports). Your Firebase **Google Cloud project ID** may still be an older id (e.g. `gathr-5b405`) because project IDs cannot be renamed. To use a new id, create a Firebase project and run `flutterfire configure`.
